Book & Author Management System (Laravel)
Project Overview

This project is a Laravel-based backend application that manages Authors and their Books.

It provides CRUD operations for both Authors and Books with proper Eloquent relationships and request validation.

Each Author can have multiple Books (One-to-Many relationship).

Features

Author Management (CRUD)

Book Management (CRUD)

Author → Books relationship

Request validation

RESTful API structure

MySQL database

Tech Stack

PHP 8+

Laravel 10+

MySQL

Composer

Postman (for API testing)
